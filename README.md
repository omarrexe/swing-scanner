# 🎯 Sniper Mode — Swing Trading Scanner

**Quality over Quantity. One Perfect Pick.**

A focused swing trading scanner designed for traders with $500-$1,000 capital who want to find THE ONE best opportunity, not 10 mediocre ones.

## What It Does

- Scans 40 high-quality stocks (mega-caps, liquid, predictable)
- Finds stocks with **70%+ win probability**
- Requires **10%+ profit target**
- Checks weekly + daily trend alignment
- Requires setup to be stable for 2+ days
- Returns **ONE best trade** (not multiple picks)

## How It Works

### Win Probability Factors
1. **Trend Alignment** — All EMAs stacked bullish
2. **ADX Strength** — Trending, not choppy
3. **RSI Sweet Spot** — Rising momentum, not overbought
4. **MACD** — Bullish histogram
5. **Volume** — Institutional activity
6. **Supertrend** — Confirmation indicator
7. **Momentum** — Recent price action

### Multi-Timeframe Confirmation
- Weekly trend must be bullish
- Daily setup must be valid for 2+ days
- No flash signals or day-trading setups

### Market Regime
- Checks SPY health before recommending trades
- Warns you when market is bearish

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Usage

1. Enter your capital (default: $1,000)
2. Click "🎯 FIND THE BEST TRADE"
3. Wait for scan (30-60 seconds)
4. Get ONE perfect trade recommendation with:
   - Entry, Stop Loss, Take Profit levels
   - Position size
   - Win probability
   - Clear reasoning

## Trading Rules

1. Only trade when market is BULLISH or NEUTRAL
2. Set stop loss immediately after buying
3. Hold 1-3 weeks (don't panic sell)
4. Take profit at target OR if RSI > 75
5. Scan once per day (after market close)
